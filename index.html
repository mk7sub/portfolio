Personal Projects

ドメイン価格比較Webアプリ

ドメイン価格のコスパ比較を行うWebアプリ。WebスクレイピングはPythonとSelenium、WebAPIはPHP(Laravel)を使用して開発。改修のため現在はWSL2でのみ稼動中。

*開発環境

デプロイ環境: XServer VPS (小規模・個人開発のため、費用対効果を優先)
サーバーOS: Rocky Linux (XServer VPS上)
コンテナ化: Docker, docker-compose
Webサーバー: Nginx
Webアプリケーションフレームワーク/API: Laravel (PHP)
フロントエンド: React (予定)
CSSフレームワーク: Tailwind CSS (予定)
データ収集/整理: Python (スクレイピングライブラリとしてSelenium、Beautiful Soup)
データベース: PostgreSQL

*履歴

2021.10～ 開発開始。
2025.2～ AI駆動開発に転換のためドキュメント、ソースコードを作り直し中。

*GitHub

ソースコード: https://github.com/mk7sub/domain-price-app
設計書: https://github.com/mk7sub/docs-domain-price-app
